Traversal Reach

Definition Mean effective vocabulary per step — the average number of syllables with non-negligible transition probability from any given starting position, under the selected profile's parameter set. Excludes self-transitions (inertia).
Calculated Using
  • Max flips (edge existence)
  • Temperature (probability distribution shape)
  • Frequency weight (rarity bias)
  • Neighbour graph adjacency
  • Softmax transition probabilities
  • Reachability threshold (p > 0.001 per starting node)
Not Influenced By
  • Seed
  • Walk count or walk steps
  • Sample size
  • Runtime randomness
Interpretation
  • Smaller reach — at each step, fewer effective choices, higher cohesion
  • Larger reach — at each step, more choices, broader exploration
  • Neither is "better" — both are stylistic regimes
Graph vs Thermodynamic Pure graph reachability (BFS with max_flips) counts all structurally connected nodes. Thermodynamic reach applies the profile's temperature and frequency weight to compute actual transition probabilities per starting node, then averages across all positions. This is why dialect and goblin — both max_flips=2 — show different reach values.
